# Glossário

| Termo | Significado |
|-------|-------------|
| Comprovai | Nome do produto — sistema de despesas corporativas com repasse via nota de débito |
| Nota de débito | Documento fiscal/administrativo gerado para repassar custo de despesa de projeto ao cliente, com PDF de comprovantes anexados |
| Piloto | Primeiro cliente de teste do produto ([[people/consuldata]]) |
| RLS | Row Level Security (Supabase/Postgres) — controle de acesso a nível de linha |
| Aprovador | Papel do sistema: gestor/gerente/diretor que aprova ou reprova despesas lançadas |
